\#  Yatra



Yatra is a full-stack travel web application developed to manage user authentication, dashboards, and travel-related informational pages.  

The project demonstrates the integration of frontend technologies with a JavaScript-based backend server and a MySQL database.



---



\##  Features

\- User login and authentication

\- User dashboard

\- Travel-related informational pages

\- Backend server handling requests

\- MySQL database integration

\- Clean and modular project structure



---



\## Tech Stack



\### Frontend
